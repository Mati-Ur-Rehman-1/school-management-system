# School Management System (Python)

## Description
This repository contains a School Management System implemented in **three versions**:

1. **CLI Version** – Simple command-line interface using Python lists.  
2. **Database Version** – Command-line interface with PostgreSQL database integration.  
3. **Flask Version** – Web application using Flask framework with HTML templates.

This project demonstrates progression from simple CLI applications to database integration and finally a web application.

---

## Project features 

### CLI Version 
- Add new students 
- Show individual student by roll number 
- Update student information 
- Remove students 
- Display all students 


### Database Version 
- All CLI version features
- Data stored in PostgreSQL database
- Persistent storage (data remains after closing the app)

 
### Flask Version 
- Web interface for all CRUD operations 
- HTML templates for student list and forms 
- Easy to use and visually organized
