# School Management System (Python)

## Description
This repository contains a School Management System implemented in **two versions**:

1. **CLI Version** – Simple command-line interface using Python lists.  
2. **Database Version** – Command-line interface with PostgreSQL database integration.  

This project demonstrates progression from a basic CLI application to a database-driven system.

---

## Project Features

### CLI Version
- Add new students
- Show individual student by roll number
- Update student information
- Remove students
- Display all students

### Database Version
- All features of the CLI version
- Data stored in PostgreSQL database
- Persistent storage (data remains after closing the application)
